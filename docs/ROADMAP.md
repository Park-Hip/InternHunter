# Roadmap

## Near Term

- Clarify the canonical ingestion entry point.
- Remove stale imports and duplicate orchestration paths.
- Expand test coverage around crawler extraction and repository writes.
- Document current search and resume matching behavior more precisely.

## Planned

- Dedicated search API.
- Dedicated resume upload and matching API.
- Alembic migrations.
- Better monitoring and failure visibility.
- More fixture-based scraper tests.

## Long Term

- Natural-language search on top of the vector store.
- Resume-aware recommendations.
- A cleaner separation between ingestion, extraction, and query serving.
